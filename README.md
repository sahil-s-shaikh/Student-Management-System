# Student Management System

## Project Overview
Student Management System is a web application developed using ASP.NET Core Web API, Entity Framework Core, SQL Server, HTML, CSS, and JavaScript.

The application allows users to:

- Add new students
- View student records
- Update student details
- Delete student records
- User login and authentication
- Store data in SQL Server database

## Technologies Used

- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- HTML
- CSS
- JavaScript

## Features

- Student CRUD Operations
- Authentication and Login
- Database Integration
- REST API Architecture
- Responsive User Interface

## Project Structure

- Controllers
- Models
- Data
- Migrations
- wwwroot

## Database

SQL Server is used as the backend database.

Run migrations before starting the project:

```bash
dotnet ef database update
```

## How to Run

1. Clone the repository
2. Open in Visual Studio
3. Update connection string in appsettings.json
4. Run database migrations
5. Start the application
